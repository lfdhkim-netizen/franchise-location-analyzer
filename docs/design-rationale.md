# Design Rationale — Franchise Location Analyzer

이 문서는 스킬 설계의 근거를 정리한다. 사용자(외식업 프랜차이즈 본사 관리자, 프리미엄 국밥 15,000원, 타겟 직장인)의 업무 컨텍스트를 그대로 반영했다.

---

## 1. 왜 멀티 페르소나 구조인가

신규점 개발 의사결정은 단일 관점으로 판단할 수 없다. **상권·재무·운영·법무·고객**이 모두 달리 본다. 현실에서도 본부 개발본부장은 부동산 평가, 운영 컨설턴트, 재무팀 의견을 모두 받아 종합 결재한다.

스킬도 같은 구조를 모사한다. 8인 페르소나가 각자 점수와 1–2 문단 코멘트를 내고, 마지막에 본부 의사결정자가 종합 판정.

**왜 8명인가**: 외식 업계 신규점 결재서에 등장하는 의견란을 분해한 결과 7–9개 관점이 일반적. 8명은 부족하지도 과하지도 않은 균형점.

---

## 2. 왜 이 6단계인가

| 단계 | 빠지면 생기는 문제 |
|---|---|
| 상권 분석 | 배후수요 없이 입지 판단 불가 |
| 타겟 적합도 | 15,000원 단가가 받아들여지는 상권인지 모름 |
| 경쟁 분석 | 카니발리제이션·차별화 평가 불가 |
| 재무 시뮬레이션 | "느낌"으로 판단하게 됨 — BEP·회수기간 없으면 결재 불가 |
| 리스크 분석 | 임대·계약 독성조항이 사후에 발견되면 매장 폐점까지 감 |
| 종합 판정 | 정량 점수를 의사결정 언어(GO/HOLD/NO-GO)로 번역해야 본부가 움직임 |

---

## 3. 직장인 타겟 특화 분석 (사용자가 제공한 컨텍스트 반영)

### 3.1 점심수요 정량화
- 반경 300m / 500m / 1km **오피스 빌딩 임직원 수** 추정
- 점심 외식률 65–75% × 한식 선택률 → 잠재 일 점심 고객 수
- **피크 90분(11:30–13:00) 회전율** 시뮬레이션: 좌석수 × 회전 가능 횟수 × 결제율

### 3.2 객단가 시뮬레이션
- 기본 국밥 15,000원 + 사이드(공깃밥/계란/김치추가) + 음료/주류
- 점심 객단가 ≈ 16,000–17,500원
- 저녁 객단가 ≈ 18,000–22,000원 (회식 시 주류 포함)

### 3.3 회식·저녁 수요
- 점심만으로는 임대료 못 견디는 상권이 많음 → 저녁 매출 비중을 반드시 추정
- 법인카드 결제 가능 매장 여부 (B2B 영업 잠재력)
- 월·금 회식 피크 패턴

### 3.4 식대 보조 한도
- 일반 대기업 식대 보조 8,000–12,000원 → 자기부담 3,000–7,000원
- CBD 오피스(여의도·강남·종로): 식대 보조 ≥ 10,000원 비율 ↑ → 15,000원 수용성 ↑
- 오피스 외곽: 식대 보조 ≤ 8,000원 → 점심 단가 12,000–14,000원이 한계선

## 4. 15,000원 프리미엄 단가 특화 분석

### 4.1 단가 수용성 매트릭스

| 상권 등급 | 점심 평균단가 | 15,000원 수용성 |
|---|---|---|
| CBD (강남·여의도·광화문) | 11,000–14,000원 | 🟢 높음 (+α 메뉴 가능) |
| 오피스 외곽 (마곡·구로·판교 일부) | 9,000–12,000원 | 🟡 중간 (단가 조정 필요) |
| 혼합 (성수·홍대 일부) | 10,000–13,000원 | 🟡 중간 (저녁 매출 의존) |
| 주거·학원가 | 8,000–11,000원 | 🔴 낮음 (NO-GO 가능성↑) |

### 4.2 차별화 포지셔닝
- 기존 국밥집 평균 단가 8,000–12,000원
- 15,000원 = 프리미엄 카테고리 → **이유**가 명확해야 함
  - 한우/한돈 등 원재료
  - 인테리어·서비스
  - 점심·저녁 동시 어필 가능한 격
- 차별화 부족 시 직장인이 "더 싼 곳에 익숙해서" 이탈

---

## 5. 프리미엄 국밥 특수 리스크

### 5.1 시즌별 매출 변동
- 국밥·국물요리는 여름철(6–8월) 매출 -20~30% 일반적
- 보수 시나리오에 반드시 반영
- 완화 방안: 여름 한정 메뉴(콩국수, 냉면, 비빔), 에어컨 강화 마케팅

### 5.2 환기·냉방 시설
- 국물·김 처리: 후드·환기 시스템 별도 투자 (1,500–3,000만원)
- 임대물건 검토 시 기존 환기시설 유무 확인
- 없으면 인테리어 비용에 추가 반영

### 5.3 식자재 신선도 동선
- 사골/한우 등 매일 새벽 입고 필요
- 도매시장(가락·마장) 접근성, 새벽 입고 차량 주차
- 식자재 보관 냉장·냉동 용량

### 5.4 인력 수급
- 점심 피크 집중 → 점심 한정 알바 모집 필요
- 인근 알바 시급 시세 (CBD vs 외곽 차이)
- 야간 운영 시 인건비 부담

---

## 6. 판정 임계값 근거

| 판정 | 기준 | 근거 |
|---|---|---|
| GO | 평균 ≥ 75 + 리스크 ≤ 30 | 외식업 신규점 첫 12개월 흑자전환률 통계 기준 |
| HOLD | 평균 60–74 | 조건 충족 시 GO 가능 → 협상 여지 |
| NO-GO | 평균 < 60 또는 리스크 > 50 | 결재 통과 어려움 + 폐점 리스크 |

---

## 7. 향후 개선 아이디어

1. **공공 데이터 연동**: 소상공인시장진흥공단 상권정보, 서울 열린데이터광장 API → 자동 인구·매출 추정
2. **이미지 입력**: 점포 사진(외관·내부) → 가시성·인테리어 평가
3. **경쟁점 자동 수집**: 카카오맵·네이버지도 API → 반경 500m 한식점 자동 매핑
4. **시계열 모니터링**: 개점 후 6/12개월 실적 vs. 예측 비교 → 모델 보정
5. **자매 스킬**:
   - 메뉴 단가 적정성 분석
   - 가맹 상담 시나리오 자동화
   - 본부 월간 점포 운영보고서
   - 슈퍼바이저 점검 체크리스트
